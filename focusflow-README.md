# FocusFlow:
A simple Android study companion app built to practice mobile development fundamentals.
You can plan study tasks, track focus sessions, browse study resources, and manage your profile.
# Features:
- Dashboard with greeting, daily tasks, study streak, and weekly study minutes
- Study Planner (add tasks by subject, choose a study mode)
- Focus Mode (Pomodoro-style timer and background music player)
- Resources (quick links by subject + custom search)
- Profile (view saved data, clear all data)
- In-app web browser for searching study resources
# How to Run:
- Open Android Studio
- Open this project folder
- Connect an Android device or start an emulator
- Click Run
- Use the Dashboard to navigate to each feature
# Requirements:
- Android Studio
- Android SDK
- Android device or emulator
# Concepts Practiced:
- Android Activities and Intents
- SQLite Database (SQLiteDatabase, Cursor)
- SharedPreferences for saving user data
- ListView and ArrayAdapter
- CountDownTimer and MediaPlayer
- WebView for in-app browsing
- UI Components: Spinner, RadioGroup, EditText, Button
